1. Current Architecture Issues
* API Key Exposure: The Gemini API key is currently exposed in the browser (via process.env.API_KEY)
--> Redesign so that the Gemini API key is not exposed in the browser.
* CORS Issues: Direct browser-to-API calls may face CORS restrictions
--> Transition to do API calls from the server.
* No Data Persistence: No tracking of student progress or results
--> Track student progress in the Backend.
